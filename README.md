# mandikal.com

Personal website for [http://mandikal.com/](http://mandikal.com/) built with [Jekyll](http://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

## Requirements

```sh
gem install github pages
```

## Usage

```sh
jekyll serve
```
